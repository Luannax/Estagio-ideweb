Bem-vindo as configurações do Web Speech!
Comandos básicos para controlar o leitor de tela Web Speech. A partir desse menu, você pode configurar o Web Speech, obter ajuda e acessar outras funções do mesmo. Aqui estão alguns exemplos:
  • Para ler o nome do site, pressione a tecla L.
  • Para mutar o leitor de tela, pressione a tecla P.
  • Para continuar a leitura da onde parou, pressione a tecla R.
  • Para navegar na Ide Web, pressione a tecla TAB.
